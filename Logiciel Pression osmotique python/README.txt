Ce logiciel permet de déterminer simplement la pression osmotique à partir 
de caractéristiques simples :
-i : le nombre d'ions en solution
-T : la température de la solution
-C : la concentration des ions dans la solution

Modules python requis :
-Tkinter
-numpy
-math
-matplotlib.pyplot

-------------

This software allows you to easily determine the osmotic pressure from
simple characteristics :

-i : the number of ions in solution
-T : the temperature of the solution
-C : the concentration of ions in the solution

Required python modules:

-Tkinter
-numpy
-math
-matplotlib.pyplot
